# MuleSoft GraphQL API

MuleSoft is an integration and API management platform. The Anypoint Platform API covers API design and publishing, CloudHub deployment, API analytics, access management, exchange artifacts, and Runtime Manager for integration apps.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from MuleSoft REST API
**Documentation:** https://docs.mulesoft.com/
