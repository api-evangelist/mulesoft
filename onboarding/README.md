# Programmatic API Onboarding — MuleSoft Anypoint

A single-file, zero-dependency Node.js (18+) CLI that reproduces SoundCloud's
`sc-api-auth.mjs` pattern for MuleSoft Anypoint: register an application / obtain credentials
programmatically instead of clicking through a dashboard, so agents and developers
can onboard at the command line.

- Script: [`mulesoft-anypoint-api-auth.mjs`](mulesoft-anypoint-api-auth.mjs)
- Run `node mulesoft-anypoint-api-auth.mjs --help` for usage and the required environment variables.
- Story / rationale: https://apievangelist.com/2026/07/08/mulesoft-anypoint-and-programmatic-api-onboarding/

Part of the API Evangelist "Programmatic API Onboarding for the Agentic Moment" series.
